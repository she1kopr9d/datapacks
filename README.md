## Настройка перед созданием датапака

### Установка папки майнкрафта

```bash
make select_dir
```

### Выбор версии

```bash
make select_version
```

### Выбор мира

```bash
make select_world
```

## Создание датапака

### 1. Создайте новый датапак

```bash
make create_datapack
```

### 2. Введите название датапака

### 3. Выберите датапак

```bash
make select_datapack
```

### Теперь вы можете разрабатывать датапак и сразу встраивать его в свой мир

## Сборка датапака

- Если вы сначала настроили окружения, а также выбрали датапак

### Сборка датапака в папку мира

```bash
make build
```
